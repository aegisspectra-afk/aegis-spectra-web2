# 📋 עמודים חסרים באתר Aegis Spectra

## ✅ עמודים קיימים:
- `/` - דף הבית
- `/about` - אודות
- `/services` - שירותים
- `/blog` - בלוג
- `/blog/[id]` - פוסט בודד
- `/product/[sku]` - עמוד מוצר בודד
- `/user` - דשבורד משתמש
- `/admin` - דשבורד אדמין
- `/admin/products` - ניהול מוצרים
- `/privacy` - מדיניות פרטיות
- `/terms` - תנאי שירות

## ❌ עמודים חסרים (לפי סדר עדיפות):

### 🔴 עדיפות גבוהה:

1. **עמוד צור קשר** (`/contact`)
   - טופס יצירת קשר מלא
   - Google Maps integration
   - פרטי קשר (טלפון, אימייל, כתובת)
   - שעות פעילות
   - Social Media links
   - WhatsApp integration
   - שדות: שם, טלפון, אימייל, נושא, הודעה

2. **עמוד הצעות מחיר** (`/quote`)
   - בחירת סוג שירות (סייבר/פיזי/משולב)
   - בחירת Package
   - פרטי הנכס (גודל, מיקום)
   - דרישות מיוחדות
   - תקציב משוער
   - Upload קבצים (תוכניות, תמונות)
   - חישוב אוטומטי של הערכת מחיר

3. **עמוד מוצרים כללי** (`/products`)
   - רשימת כל המוצרים
   - סינון לפי קטגוריה (Home, Business, Enterprise)
   - חיפוש מוצרים
   - מיון לפי מחיר/פופולריות
   - Grid/List view toggle
   - קישורים לעמודי מוצר בודדים

4. **עמוד Checkout** (`/checkout`)
   - סיכום הזמנה
   - פרטי לקוח
   - שיטת תשלום (PayPal)
   - הערכת משלוח
   - תנאי שירות
   - כפתור תשלום

5. **עמוד Success** (`/checkout/success`)
   - אישור תשלום מוצלח
   - מספר הזמנה
   - פרטי הזמנה
   - קישור למעקב הזמנה
   - CTA להמשך

### 🟡 עדיפות בינונית:

6. **עמוד Testimonials** (`/testimonials`)
   - המלצות לקוחות
   - תמונות לקוחות
   - דירוגים (5 כוכבים)
   - Carousel/Grid layout
   - פילטר לפי קטגוריית שירות

7. **עמוד FAQ נפרד** (`/faq`)
   - שאלות נפוצות מורחבות
   - קטגוריות (מוצרים, שירותים, תשלום, תמיכה)
   - חיפוש בשאלות
   - Accordion/Collapsible sections
   - קישורים לנושאים קשורים

### 🟢 עדיפות נמוכה:

8. **עמוד Gallery/Portfolio** (`/gallery` או `/portfolio`)
   - תמונות פרויקטים
   - Lightbox לתצוגה
   - פילטר לפי סוג פרויקט
   - לפני/אחרי
   - Case studies
