# 🗄️ Database Schemas - Aegis Spectra

תיקייה זו מכילה את כל קבצי ה-SQL Schema של הפרויקט.

## 📁 קבצים

### `complete-schema.sql` ⭐
**הקובץ הראשי** - מכיל את כל הסכמות ביחד. הרץ את זה ב-Netlify SQL Editor.

### סכמות נפרדות:
- `schema.sql` - סכמה בסיסית
- `products-schema.sql` - סכמת מוצרים
- `inventory-schema.sql` - סכמת מלאי
- `reviews-schema.sql` - סכמת ביקורות ודירוגים
- `loyalty-schema.sql` - סכמת נאמנות ותגמולים
- `support-schema.sql` - סכמת תמיכה טכנית
- `advanced-orders-schema.sql` - סכמת הזמנות מתקדמת
- `vendors-schema.sql` - סכמת מולטי-וונדור

### שאילתות עזר:
- `CHECK_USER.sql` - בדיקת משתמשים
- `UPDATE_PASSWORD_FINAL.sql` - עדכון סיסמה
- `UPDATE_PASSWORD.sql` - עדכון סיסמה (גרסה ישנה)

## 🚀 שימוש

### התקנה ראשונית:
```sql
-- הרץ את זה ב-Netlify Dashboard → Database → SQL Editor
-- קובץ: complete-schema.sql
```

### עדכון סכמה:
```sql
-- הרץ את הסכמה הנפרדת הרלוונטית
-- לדוגמה: inventory-schema.sql
```
